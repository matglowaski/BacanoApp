# 🐸 The Wednesday Oracle (Bacano App)

A fun Android application developed in **Kotlin** that celebrates the "Wednesday" internet meme culture. The app automatically detects the day of the week to provide a dynamic and contextual user experience.

## 🚀 Features

- **Real-time Date Detection:** The app checks the system calendar to toggle its interface.
- **Dynamic Interface:** - **Regular Days:** Displays a "waiting" screen with a frog notifying the user that Wednesday hasn't arrived yet.
  - **Wednesdays:** Unlocks the "Oracle," allowing the user to draw a random "Bacano" (frog character) representing their mood for the day.
- **Randomizer Logic:** Randomly selects an item from a custom list of frog characters and memes.

## 🛠️ Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **IDE:** Android Studio
- **Layout:** XML (ConstraintLayout) with Edge-to-Edge support for modern displays.
- **Key Components:**
- `Calendar` for date-based logic.
- `ViewCompat` for handling system window insets.
- `Activity` with dynamic `setContentView` switching.

## 💡 Key Learnings

This project was essential for mastering core Android concepts:
1. **Activity Lifecycle:** Deep dive into the `onCreate` method.
2. **Multiple Layout Management:** Handling different XML layouts within a single Activity.
3. **Debugging:** Solving ID mismatches and NullPointerExceptions.
4. **UI/UX Logic:** Applying programming logic to manipulate visual components in real-time.

- **Original Concept:** Based on the classic *"It is Wednesday, my dudes"* internet meme.

# BacanoApp
# 🐸 O Oráculo de Quarta-Feira (Bacano App)

Um aplicativo Android divertido desenvolvido em **Kotlin** que celebra a cultura dos memes de quarta-feira. O app detecta automaticamente o dia da semana e oferece uma experiência dinâmica para o usuário.

## 🚀 Funcionalidades

- **Detecção de Data:** O app verifica o calendário do sistema para mudar sua interface.
- **Interface Dinâmica:** - **Nos dias comuns:** Exibe uma interface de "espera" com o Sapo avisando que ainda não é quarta-feira.
- **Às quartas-feiras:** Libera o "Oráculo", permitindo que o usuário realize um sorteio para descobrir qual "Bacano" o representa no dia.
- **Lógica de Sorteio:** Seleciona aleatoriamente um item de uma lista personalizada de personagens (Sapinhos).

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** [Kotlin](https://kotlinlang.org/)
- **IDE:** Android Studio
- **Layout:** XML (ConstraintLayout) com suporte a Edge-to-Edge (bordas infinitas).
- **Componentes:** - `Calendar` para lógica de data.
- `ViewCompat` para ajuste de insets do sistema.
- `Activity` com troca dinâmica de `setContentView`.

## 💡 Aprendizados

Este projeto foi fundamental para consolidar conceitos como:
1. Ciclo de vida da Activity (`onCreate`).
2. Manipulação de múltiplos layouts XML no mesmo arquivo de código.
3. Tratamento de erros de ID e referências nulas no Android.
4. Lógica de programação aplicada a componentes visuais.
